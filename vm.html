# Which virtualization software to use for creating VMs? These are my preferences: * Windows: Microsoft hyperv is greater than (>) VirtualBox * Linux: virt-manager + QEMU + KVM > VirtualBox # On Linux, is virt-manager + QEMU + KVM better than VirtualBox? On Linux, virt-manager + QEMU + KVM is better than VirtualBox for me. Reasons: 1. virt-manager + QEMU + KVM is the native Linux virtualization technology. 2. virt-manager + QEMU + KVM is faster. Has support for virtio hardware. 3. VirtualBox Linux kernel mdules are not part of the Linux project.
